Paleta

Color CÚSPIDES

Una paleta corta y con carácter: verde de montaña como base sólida y profesional, rojo como acento de urgencia/acción, y lima como color de contraste para botones e íconos. Tocá cualquier swatch para copiar su hex.

Type system

Tipografías Y ESCALA

Tres familias, tres roles muy claros: Panton Rust para golpes de efecto (títulos grandes, acentos), Bogart Trial para el tono cálido/manuscrito (subtítulos, citas), e Inter para todo el texto funcional.

Habitar la montaña

Panton Rust — display / acento

Mayúsculas, peso 800. Se usa en títulos de impacto, palabras clave dentro de un título y badges numéricos grandes.

font-family: var(--font-block);

la montaña

Bogart Trial — script

Peso 700. Aporta el registro más humano: subtítulos de hero, títulos de sección, citas y nombres de tarjetas.

font-family: var(--font-script);

Creamos programas de aventura a tu medida con el conocimiento y la experiencia de toda una vida dedicada a las montañas.

Inter — texto funcional

Variable, de 100 a 900. Todo el cuerpo de texto, navegación, botones, labels y listas.

Espaciado Y RADIOS

El padding lateral de todas las secciones responde a un único token fluido, y los radios de borde son consistentes según el tipo de elemento: chico para botones internos, medio para cards, y pill completo para botones y badges.
